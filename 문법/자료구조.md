# list

### 배열 초기화
```python
# 1차원 배열
board = [0] * m

# 2차원 배열
board = [[0] * m for _ in range(n)]
```

### 배열 복사
```python
# 1차원 배열
list_b = list_a[:]

# 2차원 배열
list_b = [item[:] for item in list_a]
```

# set

set이 list보다 속도가 빠르다.

### 원소 추가
```python
s = set()

s.add(1)
s.update([1, 2, 3])
```
값을 여러개 추가할 때는 list형이나 tuple형으로 update()   
인덱싱 하려면 list나 tuple형으로 변환

### 교집합, 합집합, 차집합

```python
s1 & s2
s1 | s2
s1 - s2
```
