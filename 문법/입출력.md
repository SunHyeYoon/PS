# 입력

```python
import sys; input = sys.stdin.readline

input = input().split()
```

앞으로 input()은 해당 함수를 의미
input()은 문자열 리스트를 반환한다.

### 정수 리스트로 입력
```python
n, m = map(int, input().split())
```

### 문자열을 한 줄씩 입력
```python
text = input().rstrip()
```
rstrip()을 하지 않으면 개행문자(\n)도 함께 입력된다.

# 출력

### unpacking으로 원소 꺼내기

```python
list = [1, 2, 3]
print(*list)

# 1 2 3
```

```python
list = [1, 2, 3]
print(*list, sep = '\n')

# 1
# 2
# 3
```

```python
list = [1, 2, 3]
print(*list)

# 1 2 3
```

```python
dict = {'key1' : 1, 'key2' : 2, 'key3' : 3}
print(*dict.values())
print(*dict.keys())

# 1 2 3
# key1 key2 key3
```
